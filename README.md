# Python Torrent App

A simple command-line torrent-like file downloader and sharing tool built in Python.

## Features
- Download torrents using magnet links or .torrent files (using libtorrent)
- Simple P2P file sharing simulation
- CLI interface
- Progress tracking

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py --help
```

## Note
This is an educational/demo project. For production use, consider security and legal aspects of torrenting.
